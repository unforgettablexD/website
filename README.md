# Personal Website

This is a standalone static site for Arnav Kumar's PhD outreach and research identity.

## What it is

- one-page research-focused site
- designed to work on GitHub Pages
- intentionally minimal, fast, and easy to update

## What to put here

- short bio
- research focus
- selected papers
- selected projects
- selected GitHub repositories
- CV link
- contact links

## How to use

Open `index.html` directly in a browser, or publish the `website/` folder to GitHub Pages.

## Suggested deployment

1. Create a GitHub repository for the site.
2. Push the contents of `website/`.
3. Enable GitHub Pages on the repository.
4. Add a custom domain later if you want one.
